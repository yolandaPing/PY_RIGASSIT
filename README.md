# PY_RIGASSIT
Why Should I use this ?
=======================
``This is a binding tool assistant that can help you improve binding efficiency.``

Maya绑定助手，内部有权重模块；copy blendShape修型， weight模块；驱动模块；SDK拷贝和镜像模块；动力学绑定；拉链绑定等等
